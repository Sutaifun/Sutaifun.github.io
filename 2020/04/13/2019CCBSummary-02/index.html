A.FizzBuzz

常见问题

  • 分支条件不充分（这个是假想问题。本地可以排除，所以OJ上不会体现）

错误示范

n = int(input())
data = []
for i in range(n):
    x = int(input())
    data.append(x)
for a in data:
    if a%3 == 0:
        print('fizz')
    elif a%5 == 0:
        print('buzz')
    elif a%15 == 0:
        print('fizzbuzz')
    else:
        print(a)
#这样的话可以被15整除的都会输出fizz而不是fizzbuzz

正确答案

n = int(input())
data = []
for i in range(n):
    x = int(input())
    data.append(x)
for a in data:
    if a%3 == 0 and a%5 != 0:
        print('fizz')
    elif a%5 == 0 and a%3 != 0:
        print('buzz')
    elif a%15 == 0:
        print('fizzbuzz')
    else:
        print(a)
#判断条件要更充分

优化解法

n = int(input())
data = []
for i in range(n):
    x = int(input())
    data.append(x)
for a in data:
    if a%15 == 0:
        print('fizzbuzz')
    elif a%5 == 0:
        print('buzz')
    elif a%3 == 0:
        print('fizz')
    else:
        print(a)
#先判断15就没有任何问题啦！不妨想想为什么

奇怪的解法

n = int(input())
for i in range(n):
    x = int(input())
    flag = True
    if x%3 == 0:
        print('fizz', end='')
        flag = False
    if x%5 == 0:
        print('buzz', end='')
        flag = False
    if flag:
        print(x, end='')
    print()
#这个是我写的解法啦，有点小奇妙但是并没有简化多少，所以专门开了一栏
#顺带提一下，OJ系统上检测输出数据的时候是不受输入数据影响的
#所以输出输入穿插进行是可以的

B.对话过滤

常见问题

  • 使用ASCII码完成大小写字母匹配，但忽略了其他符号的影响
  • 对字母和符号一视同仁，顺便删去了重复的符号
  • 对字符串通过下标访问时访问越界

错误示范

s=str(input())
ns=''
for i in range(0,len(s)):    
    if s[i].isalpha()==True:
        if ord(s[i])==ord(s[i-1]) or ord(s[i])==ord(s[i-1])+32 or ord(s[i])==ord(s[i-1])-32:
            pass
        else:
            ns+=s[i]
    else:
        ns+=s[i]
print(ns)
#注意到最长的一行使用了ASCII码来判断字母是否重复
#并且使用+-32的方式将大小写字母全部考虑到了
#但是需要注意的是'!'和'A','#'和'C'等字符也是ASCII码相差32的
#因此会删除一些跟在某一些特定符号后的特定字母

s=input()
ns=s[0]
for i in range(1,len(s)):
    if str.upper(s[i])!=str.upper(s[i-1]):
        ns=ns+s[i]
print(ns)
#思路很好，但是这样不仅删去了重复的字母，还删去了重复的非字母字符

正确答案

s=input()
ns=s[0]
for i in range(1,len(s)):
    if (not str.isalpha(s[i])) or (str.upper(s[i])!=str.upper(s[i-1])):
        ns=ns+s[i]
print(ns)

优化解法

#按理说也不算优化解法，只是提供另一个思路
#同学们基本上都是通过下标将一个字符与上一个字符比较
#其实也可以创建一个变量用于存储上一个字符
s = input()
ans = ''
pre = ''
for i in s:
    if i.isalpha():
        if i.lower() == pre.lower():
            continue
    pre = i
    ans = ans + i
print(ans)
#注意这里pre就是用来存储上一字符的变量
#注意这里continue的用法

C.Cantor表

常见问题

  • 题目要求适用空格，却使用了制表符\t隔开
  • 换行的时候使用了print('\n')

错误示范

n=int(input())
for i in range(1,n+1):
    for j in range(1,n+2-i):
         print('%d/%d'%(i,j),end='\t') 
    print()
#题目要求空格但是使用了制表符'\t'

n=int(input())
for i in range(1,n+1):
    for j in range(1,n+2-i):
         print('%d/%d'%(i,j),end=' ') 
    print('\n')
#要注意print()函数自带一个end='\n'，需要仅换行的时候使用不带参数的print()即可

正确答案

n=int(input())

    for j in range(1,n+2-i):
         print('%d/%d'%(i,j),end=' ') 
    print()

D.新数根的计算

常见问题

  • 边界条件设置有问题，会输出10
  • 循环变量的赋值时机（包括赋初始值和循环过程中赋值）把握不清楚
  • 在比较的时候没有把握好变量类型

错误示范

n = str(input())
while int(n) > 10:
    b = 1
    for i in n :
        if i != '0':
            b = b * int(i)
        else:
            continue
    n = str(b)
print(n)
#如果中间计算出了10这个数字，会直接跳出循环，导致结果出错

n = str(input())
while int(n) >= 10:
    b = 1
    for i in n :
        if i != 0:
            b = b * int(i)
        else:
            continue
    n = str(b)
print(n)
#if判断的时候，i是一个字符串，所以一定不会和0相等

a=input()
m=1
while len(a)>1:
    for i in a:
        if i == '0':
            continue
        m=m*int(i)
    a=str(m)
print(a)
#m作为for循环的循环变量，应在for前初始化
#所以应该写在while里

正确答案

num = input()
while int(num) >= 10:
#while len(num) > 1:
    product = 1
    for i in num:
        if i != '0':
            product *= int(i)
    num = str(product)
print(num)
#通过数值大小或者字符串长度判断是否跳出循环都是正确思路

E.第k对角线（选做）

  • 做的人实在太少了
  • 一个比较常见的问题是题意理解不清
  • 所以直接给AC代码了

正确答案

n = int(input())
k = int(input())
vice = int(input())
for i in range(1, n+1):
    for j in range(1, n+1):
        if vice:                    #如果要画副对角线
            if k==abs(j+i-n-1):     #这里是判断(i,j)在不在n阶方阵的第k副对角线上
                print('x', end='')
            else:
                print('*', end='')
        else:                       #如果要画主对角线
            if k==abs(i-j):         #这里是判断(i,j)在不在n阶方阵的第k主对角线上
                print('x', end='')
            else:
                print('*', end='')
    print()                         #使用print()换行

F.中国剩余定理（选做）

常见问题

  • 忽略了输出正整数的条件
  • 没有注意到除法会将数据转换为浮点数，而浮点数精度可能出现问题

正确答案

n = int(input())
con = []
prod = 1                                #记录所有除数的乘积
for i in range(n):
    x, y = map(int, input().split())
    prod *= x
    con.append((x, y))                  #每一个元素是一个元组，第一项是除数，第二项是余数
rev = []                                #存储每个除数的“逆”
for item in con:
    i = 1                               #while循环变量赋初值
    att = prod // item[0]               #注意此处使用地板除来消除python浮点数运算精度不高的问题
    while att * i % item[0] != 1:
        i += 1                          #逐一寻找“逆”
    rev.append(att * i)                 #将“逆”存起来
ans = 0                                 #下面求和用的for循环变量赋初值
for i in range(n):
    ans += con[i][1] * rev[i]           #根据公式把ak+b形式算出来
ans = ans % prod                        #取最小的可能值
if ans == 0:                            #确保输出正整数
    ans = prod
print(ans)
