第四次实验问题汇总

本次试验中需要用到栈和队列这两种数据结构。为了演示自定义类的实际应用，这篇总结的A.C.E三道题均使用了class Stack():的自定义类来存储栈结构。现给出栈的定义。 

class Stack():
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if self.is_empty():
            raise Exception('stackIsEmpty')
        else:
            return self.items.pop()

    def peek(self):
        if self.is_empty():
            raise Exception('stackIsEmpty')
        else:
            return self.items[-1]

    def is_empty(self):
        return self.items == []

    def size(self):
        return len(self.items)

A.修电脑

经典的使用堆栈处理指令的题目，只需要根据输入决定入栈还是出栈即可。
需要注意的是，题目中并没有说指令只有一个字母！所以逐字符读取和使用了.isalpha()的都是思路跑偏了，实际上不复杂。

正确答案

n = int(input())
commands = Stack()                              # 实例化
for i in range(n):
    com = input()
    if com == 'Ctrl+Z':                         # 判断指令是否为撤销
        commands.pop()
    else:
        commands.push(com)
print("/".join(commands.items))                 # 使用self.items可以直接访问items对象
# 最后输出时，可以直接访问类中创建的列表，也可以逐一进行pop操作后逆序输出。

B.老千层饼

题目的本意是使用队列来从小到大存储“千层饼”。有的同学在生成新“千层饼”并传入的时候没有将已经用过的千层饼删除，这样有两个问题。其一是如果是删除后再填进去的话，会有很多的重复；第二是即便没有重复，当n值很大的时候，列表会变得非常长，这样在访问靠后的元素的时候会非常浪费时间，从而导致TLE.
有的同学在按数位依次递减的时候，减到了0以下，产生了2100-10这样的奇怪输出。

正确答案

暴力解法，注意不优化的话会TLE，需要一定程度的优化。 

n = int(input())
if 1<=n<=12:
    print(n)
else:
    n -= 12
    i = 20
    flag = False
    while n:                                    # 这里写成 while n!=0: 更容易理解
        if flag:
            i += 9
            flag = False
        else:
            i += 1
        s = str(i)
        if ord(s[-1])-ord(s[-2]) == 1:          # 末两位递增，可以一次排除掉8个非千层饼
            flag = True
        for j in range(len(s)-1):               # 对每相邻的两位进行判断
            if abs(ord(s[j])-ord(s[j+1])) > 1:
                break
        else:                                   # 找到一个千层饼，n-1
            n -= 1
    print(i)

不难看出，如果一个“千层饼”是以“67”这种严格增序列结尾的，那么下一个“千层饼”一定是+9之后的产物，这样可以优化枚举的过程，从而在规定时间内完成输出。

C.破解虫洞

这道题目本质上是前缀表达式。由于题目保证所有数字都是一位数，所以可以安心地逐个读取字符。
所谓前缀表达式，是将运算符放到数据之前，这样不需要括号来规定运算顺序也不会产生歧义。
与之对应的有后缀表达式以及我们常用的中缀表达式。
简单来讲，中缀表达式中2 - 1使用前缀表达式表示为- 2 1，使用后缀表达式表示为2 1 -；
稍复杂的例子如下：

举例：
中缀表达式：3*(6+4*(5-3))-2
计算过程：3*(6+4*(5-3))-2 = 3*(6+4*2)-2 = 3*(6+8)-2 = 3*14-2 = 42-2 = 40

前缀表达式：- * 3 + 6 * 4 - 5 3 2
计算过程： 从后向前 依次读取入栈，读取到运算符时出栈两次并执行操作， 顺序为（顶元素 操作 次顶元素） ，然后将结果入栈。
（个人喜欢用竖杠表示栈底，意为此处不可进出元素）
先依次入栈，得到|2, 3, 5
|2, 3, 5，当前元素-，剩余未读取- * 3 + 6 * 4
出栈两次，执行- 5 3，结果为2，入栈
|2, 2，当前元素4，剩余未读取- * 3 + 6 *
入栈直至读取到运算符，此时
|2, 2, 4，当前元素*，剩余未读取- * 3 + 6
出栈两次，执行* 4 2，结果为8，入栈
|2, 8，当前元素6，入栈，剩余未读取- * 3 +
|2, 8, 6，当前元素+，剩余未读取- * 3
|2, 14，当前元素3，入栈，剩余未读取- *
|2, 14, 3，当前元素*，剩余未读取-
|2, 42，当前元素-，执行最后一次操作，得到最终结果40

后缀表达式：3 6 4 5 3 - * + * 2 -
计算过程： 从前向后 依次读取入栈，读取到运算符时出栈两次并执行操作， 顺序为（次顶元素 操作 顶元素） ，然后将结果入栈。
先依次入栈，得到|3, 6, 4, 5, 3
然后读取到-，出栈两次，执行5 3 -，结果为2，入栈
|3, 6, 4, 2，当前元素*，剩余未读+ * 2 -
出栈两次，执行4 2 *，结果为8，入栈
|3, 6, 8，当前元素+，剩余未读* 2 -
|3, 14，当前元素*，剩余未读2 -
|42，当前元素2，剩余未读-
|42, 2，最后一个操作符-，执行42 2 -，得到最终结果40

正确答案

当然，如果不了解前缀表达式的话，不容易想到从后向前读取的方法。这里提供一个从前向后读取的方法。
思路就是函数符号和数字依次入栈，当栈顶两元素均为数时，运行栈顶三个元素组成的函数并入栈。当数据读完之后，栈内仅剩一个元素，即最终结果。
```python
def f(x, y):
return (x + 42y) / (42x + y)

def g(x, y):
return (xy) / (42x - 42*y)

coms = Stack() # 实例化
s = input().replace(‘(‘, ‘’).replace(‘)’, ‘’).replace(‘,’, ‘’)
for c in s: # 删掉括号和逗号
if c.isdigit():
while coms.peek() not in ‘fg’: # 判断是否连续两个数字
temp = coms.pop() # 因为有可能运算完成之后仍然是连续两个数字
opr = coms.pop() # 所以需要用while循环
if opr == ‘f’:
temp = str(f(float(temp), float(c)))
else:
temp = str(g(float(temp), float(c)))
c = temp
if coms.is_empty(): # 当栈为空说明运算完成了，提前跳出避免报错
break
coms.push(c)
print(“%.7f”%float(coms.pop()))

```python
def f(x, y):
    return (x + 42*y) / (42*x + y)

def g(x, y):
    return (x*y) / (42*x - 42*y)

coms = Stack()                                  # 实例化
s = list(input())                               # 使用list是为了后续使用.reverse()方法
s.reverse()                                     # 逆序便于完成从后向前读取
for c in s:
    if c.isdigit():                             # 将数字入栈
        coms.push(int(c))
    elif c == 'f':                              # 遇到函数符则将栈顶两元素弹出，运算后入栈
        coms.push(f(coms.pop(), coms.pop()))    # 要注意出栈顺序
    elif c == 'g':
        coms.push(g(coms.pop(), coms.pop()))
print("%.7f"%coms.pop())

首先明确一点，括号和逗号并不能影响运算，所以我们将其排除。
然后使用样例举例:f(9, g(f(5, 2), 7))
去掉括号后变成f, 9, g, f, 5, 2, 7
逆序之后变成7, 2, 5, f, g, 9, f
然后从前向后读取。首先读到7, 2, 5，均入栈。此时栈内元素为
|7, 2, 5
然后读取到f，则出栈两次，依次得到5，2，然后施加f(x, y)函数
将函数的返回值f(5, 2)入栈。此时栈内元素为
|7, f(5, 2)
然后读到g，出栈两次得到f(5, 2)和7，施加g(x, y)函数，将返回值入栈。此时栈内元素为
|g(f(5, 2), 7)
然后读取9，入栈，栈内元素为
|g(f(5, 2), 7), 9
最后读到f，出栈两次后运算，结果入栈。此时栈内仅剩一个元素
|f(9, g(f(5, 2), 7))，即为最终结果。

## <font color=#0033FF>D.合理复习（选做）</font>
这道题又是构造类题目，代码不难，难在思维上。只需要明白可供复习的时间总长大于等于需要的时间总长就一定可以给出一个复习方案。我们只需要证明用$1$到$n$的所有正整数可以从中挑出若干不重复的整数，使之和为$\left[ 1, \frac {n \left( n+1 \right)}{2} \right]$中任意一个整数即可。  
具体构造方法如下：  
> 对于$\left[1, n\right]$中的整数，直接取自身即可；
> 对于接下来的$\left[n+1, n+\left(n-1\right)\right]$中的整数，取$n$，再从剩下的数中取一个即可；
> 对于接下来的$\left[n+\left(n-1\right)-1, n+\left(n-1\right)+\left(n-2\right)\right]$中的整数，取$n$和$n-1$，再取剩下的数中的一个；
> 以此类推即可证明以上结论。  

因此这道题就变成了找到最小的$n$，使得$\sum^{n}_{i=1} i \ge x$.

E.复杂相对分子质量（选做）

这道题目的答题思路是使用栈进行数据存储和运算。读取到元素符号则将其原子量压入，读取到左括号直接压入，读取到数字则弹出栈顶元素，乘上数字后再压入，读取到右括号则弹出至左括号，将弹出的数据全部相加再压入，最后栈内剩下的唯一元素则是最终结果。
有的同学在一开始删除了所有的括号，可能是受到了破解虫洞那道题的影响。 这道题目的括号会对最终结果产生影响，所以不能删去括号！

正确答案

c = input() + '/'                               # 为了能够录入最后一个数，在末尾添加一个符号
s = Stack()                                     # 实例化
temp = ''                                       # 用于存放数字，因为不保证数都是一位，所以需要这个变量
Ar={'H':1.007, 'C':12.011,                      # 字典，存放原子量
    'N':14.007, 'O':15.999}
for ch in c:                                    # 逐字符遍历
    if not ch.isdigit():                        # 如果不是数字，这里也可以写 if ch.isdigit() == 0:
        if temp != '':                          # temp非空说明需要将前一个原子的原子量乘以乘数
            m = s.pop()
            s.push(int(temp) * m)
            temp = ''                           # 注意清空temp
    else:                                       # 如果是数字，则暂存进temp
        temp += ch
        continue                                # 并且不用进行后续处理
    if ch == '(':                               # 左括号直接压栈
        s.push(ch)
        continue                                # 此处不写continue，后续使用elif也是可以的
    if ch in Ar.keys():                         # 如果是元素符号
        s.push(Ar[ch])                          # 则压入原子量
        continue
    if ch == ')':                               # 如果是右括号
        sum_ = 0                                # 循环变量赋初值
        while s.peek() != '(':                  # 不断弹出直至遇到左括号
            sum_ += s.pop()                     # 累加弹出的值
        s.pop()                                 # 弹出左括号
        s.push(sum_)                            # 压入总和
ans = 0
while s.is_empty() == 0:                        # 栈内剩余元素的和即为答案
    ans += s.pop()
print("%.2f"%ans)
